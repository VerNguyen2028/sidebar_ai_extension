# sidebar_ai_extension
Rảnh rỗi sinh nông nõi, tôi kết hợp với AI để tạo ra dự án này.

--- Instruction to instal ---

1. Download the release of frontend [here](https://github.com/VerNguyen2028/sidebar_ai_extension/releases/latest)
After that, put this to the browser's extensions management tools under developer mode. Anything as the same, I think so.
And you can see extension's sidebar interface, but you can use it right now. Let's do the second step below.


2. Pull source code backenđ in the repo and wait a little bit. You must run command (npm install) to get requested library and set the my own API KEY. After all, run node index.js.

--- Some other aspects ---
1. AI Model that I am using is Gemini 2.5 Flash. If you has many money, you can try models pro. Just do revise source backend and using AI.
2. So using model flash, you sometimes need to change gemini's api key. Just do revise the source code backend and ask someone that you already know someone is.
3. If you are too lazy when run the source code backend as turn on the device to do hell something. I highly recommend tools that initialize automatively such as PM2 in Window or systemd in Ubuntu, let ask AI for further instruction. About Mac, I don't know.
4. When using this software, a few error may occur:
    Error 400, 402, 403: Wrong API Key or invalid input.
    Error 404: Unfound Model AI.
    Error 429: API Key has exceed quota.
    Error 500: Error from Google Gemini AI.
    Error 503: Google Gemini AI not availability or in high demand.
